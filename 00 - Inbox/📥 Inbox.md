# 📥 Inbox — Bandeja de entrada

> [!tip] Captura primero, ordena después
> Aquí va TODO lo que llegue y no sepas dónde poner: un archivo descargado, una idea, un pendiente, algo de clase o del trabajo. Sin pensar dónde va.

## 🤖 Cómo se procesa (automático + Claude)

**1. Conexión automática (red de seguridad)**
Cada vez que se hace un commit o un pull (en cualquier compu), un *git hook* regenera los índices y **conecta solo** todo lo nuevo. Nada queda huérfano. Tú no haces nada.

**2. Clasificación inteligente (bajo pedido)**
Lo que la máquina NO puede hacer es *decidir* dónde va cada archivo (¿Dismac? ¿UPDS? ¿qué materia?). Eso lo hace Claude. Cuando tengas cosas aquí, solo abre Claude Code y di:

> **"Claude, organiza el Inbox"**

Y Claude lee cada archivo, decide a qué área/materia pertenece, lo mueve a su sitio y lo conecta con las notas relacionadas. Tú no organizas nada — solo avisas.
