PASOS PARA USAR EL PROGRAMA:
========================================

PASO 1: VERIFICAR (Opcional pero recomendado)
----------------------------------------------
1. Abra la carpeta: generador_reportes
2. Doble clic en: VERIFICAR.bat
3. Esto verificará que Python esté instalado y los archivos sean correctos
4. Si hay problemas, el script le dirá qué hacer


PASO 2: INSTALAR (Solo la primera vez)
----------------------------------------------
1. Abra la carpeta: generador_reportes
2. Doble clic en: INSTALAR.bat
3. Espere mientras se instalan las librerías:
   - python-pptx (para crear PowerPoint)
   - matplotlib (para gráficos)
   - Pillow (para imágenes)
4. Cuando termine, cierre la ventana


PASO 3: EJECUTAR
----------------------------------------------
1. Doble clic en: EJECUTAR.bat
2. Se abrirá la interfaz gráfica
3. Verá 4 pestañas con datos precargados:

   📋 PESTAÑA 1: Datos Generales
   - Mes y año
   - Nombre de la tienda
   - Objetivos ($us y Bs)
   - Ventas alcanzadas
   - Ticket promedio
   - Porcentajes de contado y minicuotas

   📊 PESTAÑA 2: Categorías
   - 5 categorías precargadas
   - Botón "Agregar Categoría" (nueva)
   - Botón "Eliminar Seleccionada"
   - Cada categoría tiene: nombre, objetivo, ventas, alcance %

   🏆 PESTAÑA 3: Top 10 SKUs
   - 10 SKUs precargados
   - Botón "Agregar SKU"
   - Botón "Eliminar Seleccionado"
   - Cada SKU tiene: código y cantidad vendida

   ⭐ PESTAÑA 4: Mystery Shopper
   - Índice general (%)
   - Atención del vendedor (%)
   - 5 puntos a mejorar con porcentajes

4. MODIFICAR DATOS (opcional):
   - Puede cambiar cualquier número o texto
   - Puede agregar o eliminar categorías/SKUs
   - Los datos están precargados del PDF original

5. GENERAR REPORTE:
   - Clic en el botón verde "GENERAR REPORTE"
   - Elegir dónde guardar el archivo
   - Sugerencia: guardarlo en la carpeta "output"
   - Nombre sugerido: Cierre_Mes_Diciembre_2024.pptx
   - Esperar 5-10 segundos
   - ¡Listo!

SOLUCIÓN DE PROBLEMAS:
========================================

❌ Problema: "Python no está instalado"
✓ Solución: Descargar e instalar desde https://www.python.org/downloads/
           Durante instalación marcar "Add Python to PATH"

❌ Problema: "No module named 'pptx'"
✓ Solución: Ejecutar INSTALAR.bat nuevamente

❌ Problema: La ventana se cierra inmediatamente
✓ Solución: Ejecutar desde CMD para ver errores:
           cd generador_reportes
           python main.py

❌ Problema: "Permission denied" al guardar
✓ Solución: Elegir otra ubicación (Escritorio o Documentos)

❌ Problema: Los gráficos no se ven
✓ Solución: Verificar que la carpeta "assets" exista
           Se crea automáticamente al ejecutar
